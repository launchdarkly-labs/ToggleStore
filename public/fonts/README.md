# Sohne Font Files

## Required Font Files

Place the following Sohne font files in this directory:

### Sohne (Sans-serif)
- `Sohne-Buch.woff2` - Regular weight (400)
- `Sohne-Kraftig.woff2` - Bold weight (700)

### Sohne Mono (Monospace)
- `SohneMono-Buch.woff2` - Regular weight (400)

## Where to Get Sohne

Sohne is a commercial typeface designed by [Klim Type Foundry](https://klim.co.nz/retail-fonts/sohne/).

You can purchase it from:
- Klim Type Foundry: https://klim.co.nz/retail-fonts/sohne/

## Font Weights Used in ToggleStore

- **400 (Regular)**: Body text, descriptions, labels
- **700 (Bold)**: Headings, product names, CTAs

## File Format

The app is configured to use WOFF2 format for optimal performance and browser compatibility.

## Fallback Fonts

If Sohne fonts are not available, the app will fall back to:
- **Sans-serif**: system-ui, -apple-system, Segoe UI, Helvetica, Arial
- **Monospace**: ui-monospace, Consolas

## Converting Font Files

If you have Sohne in other formats (OTF, TTF, WOFF), you can convert them to WOFF2 using:
- [Transfonter](https://transfonter.org/)
- [CloudConvert](https://cloudconvert.com/otf-to-woff2)
